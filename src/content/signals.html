第1章 信号与系统的基本概念

1.1 信号的分类

信号周期性判断（例1-1-2）：连续信号 $f(t) = f(t+T)$，最小正 $T$ 为周期。离散信号 $f(k) = f(k+N)$，$N$ 需为整数。

能量信号与功率信号快速判断

信号类型判断条件结论
直流信号恒定值功率信号
周期信号$f(t)=f(t+T)$功率信号
脉冲信号$|t|\to\infty$ 时幅值趋于 0能量信号
幅值有界（不趋于0）$|t|\to\infty$ 幅值不为无穷大，至少一边有限功率信号
幅值无界$|t|\to\infty$ 只要一边幅值为无穷大非能量非功率
能量与功率公式
$$E = \int_{-\infty}^{\infty} |f(t)|^2 \, dt \qquad P = \lim_{T\to\infty} \frac{1}{2T} \int_{-T}^{T} |f(t)|^2 \, dt$$ $$\text{能量信号: } 0 < E < \infty, \; P=0 \qquad \text{功率信号: } E=\infty, \; 0 < P < \infty$$

1.2 系统的分类

线性系统判断（快速判断法）

非线性系统的三个快速判据：
① 方程中有常数项
② $x$ 及 $x$ 各阶导数的次数 > 1
③ $y$ 及 $y$ 各阶导数的次数 > 1
与原方程式形式不同 → 不满足叠加性 → 非线性系统

时不变系统判断（快速判断法）

时不变系统的两个条件：
① $y$ 或 $x$ 前系数为常数
② $y$ 或 $x$ 中 $t$ 前系数为 1

因果系统判断（快速判断法）

因果性判断：代入 $t=0$（或 $k=0$）：
- $y$ 括号里数值 $<$ $x$ 括号里数值 → 非因果
- $y$ 括号里数值 $\geq$ $x$ 括号里数值 → 因果
因果系统：响应不超前于激励，没有输入就没有输出。实际系统都是因果系统；非因果系统是理想系统，不是真实系统。

例题：判断系统线性/时变/因果性（点击展开）

例：某系统由微分方程 $y'(t) + 2y(t) = x(t) + 3$ 表示，试判断该系统是否为线性系统。

解：方程中存在常数项 3，不满足齐次性 → 非线性系统
例：试判断 $y(t) = tx(t)$ 是否为时不变系统。

解：$x$ 前系数为 $t$（不是常数）→ 时变系统
例：试判断 $y(t) = x(t+1)$ 是否为因果系统。

解：当 $t=0$ 时，$y(0)=x(1)$，当前输出与将来输入有关 → 非因果系统

第2章 连续时间信号与系统的时域分析

2.1 典型连续时间信号

要求：给定信号的波形，会用基本信号写出信号的表达式。

单位冲激信号 $\delta(t)$ 的性质

性质公式说明
筛选特性$\displaystyle\int_{-\infty}^{\infty} f(t)\delta(t)\,dt = f(0)$也叫"抽样特性"
加权特性$f(t)\delta(t) = f(0)\delta(t)$广义函数等价
尺度变换$\delta(at) = \frac{1}{|a|}\delta(t)$
偶函数$\delta(-t) = \delta(t)$
注意：在有限积分区间 $(a,b)$ 内使用筛选特性时，只有当 0 在积分区间内，$\int_a^b f(t)\delta(t)\,dt = f(0)$；否则为 0。

例题：冲激函数的筛选/加权特性应用（点击展开）

例2-1-1：已知 $f(t)$ 的波形如图所示，试求 $f''(t)$，并画出其波形图。

解：利用微分性质，含有冲激函数的求导：
- 在信号跳变点处产生冲激 $\delta(t)$
- 利用加权特性 $f(t)\delta(t-t_0) = f(t_0)\delta(t-t_0)$
- 阶跃部分求导为冲激，常数部分求导为 0

2.2 信号的基本运算与时域分解

基本运算

运算说明
翻转$f(-t)$：关于纵轴翻转
尺度变换$f(at)$：$|a|>1$ 压缩，$|a|<1$ 展宽
时移$f(t-t_0)$：$t_0>0$ 右移，$t_0<0$ 左移
复合运算顺序：先翻转 → 再压缩/展宽 → 最后时移（建议时移放最后一步）

信号的时域分解

分解方式
交直流分解：$f(t) = f_D + f_A(t)$，其中 $f_D$ 为直流分量
奇偶分解：$f(t) = f_e(t) + f_o(t)$，其中 $f_e(t) = \frac{1}{2}[f(t)+f(-t)]$ 为偶分量

2.3 卷积运算

卷积积分定义
$$y(t) = f_1(t) * f_2(t) = \int_{-\infty}^{\infty} f_1(\tau) f_2(t-\tau) \, d\tau$$

卷积积分的重要性质

性质公式
交换律$f_1 * f_2 = f_2 * f_1$
分配律$f_1 * (f_2 + f_3) = f_1*f_2 + f_1*f_3$
结合律$(f_1 * f_2) * f_3 = f_1 * (f_2 * f_3)$
时移性若 $f_1*f_2 = y(t)$，则 $f_1(t-t_1)*f_2(t-t_2) = y(t-t_1-t_2)$
含冲激函数$f(t) * \delta(t) = f(t)$，$f(t) * \delta(t-t_0) = f(t-t_0)$
微积分性质
$$\frac{d}{dt}[f_1(t)*f_2(t)] = f_1'(t)*f_2(t) = f_1(t)*f_2'(t)$$ $$\int_{-\infty}^{t} [f_1(\tau)*f_2(\tau)] \, d\tau = \left[\int_{-\infty}^{t} f_1(\tau)\,d\tau\right] * f_2(t)$$
串/并联系统的冲激响应：
- 串联：$h(t) = h_1(t) * h_2(t)$
- 并联：$h(t) = h_1(t) + h_2(t)$

2.4 系统的全响应

全响应分解
$$\text{全响应} = \text{零输入响应} + \text{零状态响应}$$
冲激响应 $h(t)$：系统在零状态条件下，输入为 $\delta(t)$ 时的响应。
已知 $h(t)$，则系统对任意输入 $x(t)$ 的零状态响应为：$y_{zs}(t) = x(t) * h(t)$

第4章 连续时间信号与系统的频域分析

4.1 傅里叶级数

三角形式傅里叶级数

三角形式
$$f(t) = a_0 + \sum_{n=1}^{\infty} \left[ a_n \cos(n\omega_0 t) + b_n \sin(n\omega_0 t) \right]$$ 其中 $\omega_0 = \frac{2\pi}{T}$ 为基波频率。

指数形式傅里叶级数

指数形式
$$f(t) = \sum_{n=-\infty}^{\infty} F_n e^{jn\omega_0 t}, \quad F_n = \frac{1}{T}\int_{T} f(t)e^{-jn\omega_0 t}\,dt$$

4.2 周期信号的频谱特点

周期信号频谱的三大特点：
① 离散性：谱线是离散的，不是连续的
② 谐波性：谱线只能出现在基波频率 $\omega_0$ 的整数倍上
③ 收敛性：谱线幅度随 $n\to\infty$ 衰减到零（谐波次数越大，谱线高度越小）

单边频谱与双边频谱

单边频谱 → 双边频谱的转换规则：
- 双边幅度频谱：偶对称，$|F_n| = |F_{-n}|$
- 双边相位频谱：奇对称，$\varphi_n = -\varphi_{-n}$
- 注意表示成规范形式（余弦形式），以及两者之间的关系

例题：由单边频谱求时域表达式（点击展开）

例：已知某周期信号的单边频谱，试写出该信号的时域表达式，并画出其双边频谱。

解题步骤：
① 根据单边频谱写出余弦形式的傅里叶级数
② 转换为双边频谱：幅度偶对称，相位奇对称
③ 可进一步写成指数形式

4.3 傅里叶变换

傅里叶变换对
$$F(\omega) = \int_{-\infty}^{\infty} f(t) e^{-j\omega t} \, dt \qquad f(t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} F(\omega) e^{j\omega t} \, d\omega$$

实信号奇偶性：
- $f(t)$ 为实偶函数 → $F(\omega)$ 为实偶函数
- $f(t)$ 为实奇函数 → $F(\omega)$ 为虚奇函数

复合运算顺序：翻转 → 压缩 → 移位

4.4 采样定理与奈奎斯特取样率

奈奎斯特采样定理
$$\boxed{f_s \geq 2f_{\max}}$$ 带限信号 $f(t)$ 的最高频率为 $f_{\max}$（带宽 $W_m$），采样频率 $f_s$ 必须不小于 $2f_{\max}$ 才能从采样信号中无失真地恢复原信号。

带宽计算规则

信号运算后的带宽：
- 两信号相乘$f_1(t)\cdot f_2(t)$：带宽 = 原两信号带宽之和
- 两信号相加$f_1(t)+f_2(t)$：带宽 = 原两信号带宽中较大的
- 两信号卷积$f_1(t)*f_2(t)$：带宽 = 原两信号带宽中较小的
单位换算：角频率 $\omega$ 单位 rad/s（弧度/秒），频率 $f$ 单位 Hz（赫兹），关系 $f = \omega / 2\pi$。

例题：奈奎斯特取样率计算（点击展开）

例：设 $f(t)$ 为带限信号，带宽为 $W_m$，试分别求 $f(t)\cos\omega_0 t$ 和 $f^2(t)$ 的带宽和奈奎斯特取样率。

解：
① $f(t)\cos\omega_0 t$：由调制定理，频谱搬移到 $\pm\omega_0$，带宽 = $W_m + \omega_0$，$f_s \geq 2(W_m + \omega_0)/2\pi$
② $f^2(t)$：时域相乘 → 频域卷积，带宽 = $2W_m$，$f_s \geq 2 \cdot 2W_m/2\pi = 4W_m/2\pi$

4.5 系统频域分析 · 无失真传输 · 理想滤波器

频域分析核心：由时域卷积定理 $y(t) = x(t)*h(t) \Leftrightarrow Y(\omega) = X(\omega) \cdot H(\omega)$
正（余）弦信号输入到 LTI 系统，输出是同频率的信号，只改变了幅度和相位。
无失真传输条件
$$H(\omega) = K e^{-j\omega t_0}$$ 即 $|H(\omega)| = K$（常数），$\varphi(\omega) = -\omega t_0$（线性相位）。

5.3 系统函数 $H(s)$ 与系统稳定性

系统函数
$$H(s) = \frac{Y(s)}{X(s)} = \frac{\text{零状态响应的拉氏变换}}{\text{输入的拉氏变换}}$$ $H(s)$ 与冲激响应 $h(t)$ 是一对拉普拉斯变换对。
由 $H(s)$ 判断系统稳定性：
① 稳定系统：$H(s)$ 的所有极点均位于 $s$ 左半平面
② 临界稳定：$H(s)$ 在虚轴上（包括原点）有一阶极点，其余极点在左半平面
③ 不稳定系统：$H(s)$ 有位于 $s$ 右半平面的极点，或虚轴上有二阶以上的极点
快速判稳：对于稳定系统，分母多项式 $D(s)$ 各项系数均为正实常数，且多项式中无缺项。
对于二阶系统，只要 $D(s)$ 各项系数均为正实常数，则系统稳定。

例题：由零极点图求系统函数（点击展开）

例：已知系统的零、极点图，并且该系统阶跃响应的终值为 3，试写出系统函数的表达式。

解：
① 由零极点图写出 $H(s) = H_0 \cdot \frac{(s-z_1)\cdots}{(s-p_1)\cdots}$
② 对阶跃响应 $y(t)$ 应用终值定理：$y(\infty) = \lim_{s\to 0} s \cdot \frac{H(s)}{s} = \lim_{s\to 0} H(s) = 3$
③ 由此确定系数 $H_0$

例题：系统稳定性判断（点击展开）

例5-5-4：已知系统函数 $H(s) = \frac{k}{s^2 + (3-k)s + 2}$，求系统稳定时 $k$ 的范围。

解：系统是二阶系统，分母 $D(s) = s^2 + (3-k)s + 2$
各系数为正的条件：$3-k > 0$ 且 $2 > 0$
∴ $k < 3$ 时系统稳定

6.3 系统函数 $H(z)$ 与系统特性

系统函数
$$H(z) = \frac{Y(z)}{X(z)}$$ $H(z)$ 与单位脉冲响应 $h(k)$ 是一对 Z 变换对。
由 $H(z)$ 判断离散系统稳定性：
- 稳定：$H(z)$ 的所有极点在单位圆内（$|z|<1$）
- 临界稳定：单位圆上有一阶极点，其余在单位圆内
- 不稳定：有极点在单位圆外，或单位圆上有二阶以上极点
终值定理 vs 收敛域：
- $F(s)$ 的极点在 $s$ 左半平面 → 终值存在
- $F(z)$ 的极点在单位圆内 → 终值存在
- 两者判断方法一致，只是域不同
