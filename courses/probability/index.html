概率论部分 第1-5章

第一章 随机事件及其概率

▼

1事件间的关系与运算

  • 包含：\(A \subset B\) 表示 A 发生必导致 B 发生
  • 相等：\(A = B \Leftrightarrow A \subset B \text{ 且 } B \subset A\)
  • 和（并）：\(A \cup B\) 表示 A 与 B 至少一个发生
  • 交（积）：\(A \cap B\) 表示 A 与 B 同时发生
  • 差：\(A - B = A\bar{B}\) 表示 A 发生但 B 不发生
  • 互斥（互不相容）：\(A \cap B = \varnothing\)
  • 对立（互补）：\(A \cup \bar{A} = \Omega,\; A \cap \bar{A} = \varnothing\)
De Morgan 律（对偶律）： \(\overline{A \cup B} = \bar{A} \cap \bar{B}\)，\(\overline{A \cap B} = \bar{A} \cup \bar{B}\)

2概率的基本性质

  • \(0 \le P(A) \le 1\)
  • \(P(\Omega) = 1,\; P(\varnothing) = 0\)
  • 加法公式：\(P(A \cup B) = P(A) + P(B) - P(AB)\)
  • 对立事件：\(P(\bar{A}) = 1 - P(A)\)
  • 减法公式：\(P(A - B) = P(A) - P(AB)\)
  • 若 \(A \subset B\)：\(P(B - A) = P(B) - P(A)\)，\(P(A) \le P(B)\)

3条件概率

\(P(B|A) = \dfrac{P(AB)}{P(A)}, \quad P(A) > 0\)
  • 乘法公式：\(P(AB) = P(A) \cdot P(B|A) = P(B) \cdot P(A|B)\)
  • 推广：\(P(A_1 A_2 \cdots A_n) = P(A_1) P(A_2|A_1) P(A_3|A_1 A_2) \cdots P(A_n|A_1 \cdots A_{n-1})\)

4全概率公式与贝叶斯公式

设 \(A_1, A_2, \ldots, A_n\) 为 \(\Omega\) 的一个完备事件组（划分），即 \(A_i\) 两两互斥且 \(\bigcup_{i=1}^n A_i = \Omega\)。

全概率公式：\(P(B) = \sum_{i=1}^{n} P(A_i)\, P(B|A_i)\)
贝叶斯公式（逆概率公式）： \(P(A_i|B) = \dfrac{P(A_i)\, P(B|A_i)}{\displaystyle\sum_{j=1}^{n} P(A_j)\, P(B|A_j)}\)
【例题】某晶体管由三家工厂供货，次品率分别为 0.02, 0.01, 0.03，供货份额分别为 0.15, 0.80, 0.05。
(1) 随机取一只，是次品的概率？
(2) 取到次品，出自各厂的概率？

解：设 \(A_i\) = 第 i 厂产品，B = 次品。
\(P(B) = 0.15 \times 0.02 + 0.80 \times 0.01 + 0.05 \times 0.03 = 0.003 + 0.008 + 0.0015 = 0.0125\)
\(P(A_1|B) = \frac{0.003}{0.0125} = 0.24\)，\(P(A_2|B) = \frac{0.008}{0.0125} = 0.64\)，\(P(A_3|B) = \frac{0.0015}{0.0125} = 0.12\)

5独立性与三大概型

独立性：A、B 独立 \(\Leftrightarrow P(AB) = P(A) \cdot P(B)\)

  • 古典概型：等可能有限样本空间，\(P(A) = \frac{|A|}{|\Omega|}\)
  • 几何概型：\(P(A) = \frac{S_A}{S_\Omega}\)（测度之比）
  • 伯努利概型（n重独立试验）：\(P(X = k) = C_n^k p^k (1-p)^{n-k},\; X \sim B(n,p)\)

✎ 经典例题精讲

【例1 · 条件概率】设 \(P(A) = \frac{1}{4}\)，\(P(B) = \frac{1}{2}\)，求 \(P(B\bar{A})\)。

(1) A与B互斥时：
因 A、B 互斥，\(AB = \varnothing\)，故 \(B\bar{A} = B - AB = B\)。 \(P(B\bar{A}) = P(B) = \dfrac{1}{2}\)

(2) \(A \subset B\) 时：
因 \(A \subset B\)，\(B\bar{A} = B - A\)，所以 \(P(B\bar{A}) = P(B) - P(A) = \frac{1}{2} - \frac{1}{4} = \frac{1}{4}\)

(3) \(P(AB) = \frac{1}{9}\) 时：
\(P(B\bar{A}) = P(B - AB) = P(B) - P(AB) = \frac{1}{2} - \frac{1}{9} = \frac{7}{18}\)
【例2 · 加法公式】设 \(P(A) = P(B) = P(C) = \frac{1}{4}\)，\(P(AB) = P(BC) = 0\)，\(P(AC) = \frac{1}{8}\)，求 \(P(A \cup B \cup C)\)。

解：由三事件加法公式： \(P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(AB) - P(AC) - P(BC) + P(ABC)\) 注意 \(AB = \varnothing \Rightarrow ABC = \varnothing\)，所以 \(= 3 \times \frac{1}{4} - 0 - \frac{1}{8} - 0 + 0 = \dfrac{5}{8}\)
【例3 · 乘法公式】透镜第一次落下打破概率 \(\frac{1}{2}\)，若未打破，第二次打破概率 \(\frac{7}{10}\)，若仍未打破，第三次打破概率 \(\frac{9}{10}\)。求三次落下未打破的概率。

解：设 \(A_i\) = 第 i 次落下打破。三次未打破 = \(\bar{A_1}\bar{A_2}\bar{A_3}\)，由乘法公式： \(P(\bar{A_1}\bar{A_2}\bar{A_3}) = P(\bar{A_1}) \cdot P(\bar{A_2}|\bar{A_1}) \cdot P(\bar{A_3}|\bar{A_1}\bar{A_2})\) \(= \left(1-\frac{1}{2}\right)\left(1-\frac{7}{10}\right)\left(1-\frac{9}{10}\right) = \frac{1}{2} \times \frac{3}{10} \times \frac{1}{10} = \dfrac{3}{200}\)
【例4 · 全概率+贝叶斯（枪射击）】5支枪中2支未校正、3支已校正。校正枪中靶率 0.9，未校正枪中靶率 0.4。
(1) 任取一支射击，中靶概率？
(2) 射击未中靶，求该枪未校正的概率。

解：设 A = 枪已校正，B = 中靶。\(P(A) = 0.6\)，\(P(\bar{A}) = 0.4\)，\(P(B|A) = 0.9\)，\(P(B|\bar{A}) = 0.4\)。

(1) 全概率公式： \(P(B) = P(A)P(B|A) + P(\bar{A})P(B|\bar{A}) = 0.6 \times 0.9 + 0.4 \times 0.4 = 0.54 + 0.16 = 0.70\)

(2) 贝叶斯公式：\(P(\bar{A}|\bar{B}) = \frac{P(\bar{A})P(\bar{B}|\bar{A})}{P(\bar{B})} = \frac{0.4 \times 0.6}{0.30} = 0.8\)
【例5 · 贝叶斯（癌症筛查）】某地区肝癌发病率 0.0004。甲胎蛋白法检测：肝癌患者阳性率 99%，非肝癌者假阳性率 0.1%。某人检查阳性，求真患肝癌的概率。

解：设 B = 患肝癌，A = 检查阳性。 \(P(B) = 0.0004\)，\(P(\bar{B}) = 0.9996\)，\(P(A|B) = 0.99\)，\(P(A|\bar{B}) = 0.001\)
由贝叶斯公式： \(P(B|A) = \frac{P(B)P(A|B)}{P(B)P(A|B) + P(\bar{B})P(A|\bar{B})}\) \(= \frac{0.0004 \times 0.99}{0.0004 \times 0.99 + 0.9996 \times 0.001} = \frac{0.000396}{0.000396 + 0.0009996} \approx 0.284\)
即使检查呈阳性，真正患癌概率仅约 28.4%。这是因为肝癌发病率极低，假阳性的"基数"远大于真阳性。

第二章 随机变量及其分布

▼

1离散型随机变量

分布律：\(P(X = x_k) = p_k,\; k = 1,2,\ldots\)，满足 \(p_k \ge 0\)，\(\sum_k p_k = 1\)。

常见离散分布：0-1分布、二项分布 \(B(n,p)\)、泊松分布 \(\pi(\lambda)\) 等。

泊松定理（二项近似泊松）： 若 \(n\) 大、\(p\) 小、\(np = \lambda\) 适中，则 \(C_n^k p^k(1-p)^{n-k} \approx \dfrac{\lambda^k e^{-\lambda}}{k!}\)

2连续型随机变量

概率密度 \(f(x)\)：\(f(x) \ge 0\)，\(\int_{-\infty}^{+\infty} f(x)\,dx = 1\)

分布函数：\(F(x) = P(X \le x) = \int_{-\infty}^{x} f(t)\,dt\)

概率计算：\(P(a < X \le b) = F(b) - F(a) = \int_a^b f(x)\,dx\)

在 \(f(x)\) 的连续点处：\(F'(x) = f(x)\)

常见连续分布：均匀分布 \(U(a,b)\)、指数分布 \(Exp(\lambda)\)、正态分布 \(N(\mu, \sigma^2)\)

3随机变量函数的分布

离散型：直接列 \(Y = g(X)\) 的取值与概率。

连续型——分布函数法：

  • 先求 \(F_Y(y) = P(Y \le y) = P(g(X) \le y)\)
  • 再求导 \(f_Y(y) = F_Y'(y)\)

公式法（严格单调时）：

设 \(Y = g(X)\) 严格单调，反函数 \(x = h(y)\) 有连续导数，则 \(f_Y(y) = f_X[h(y)] \cdot |h'(y)|,\quad a < y < b\)
正态分布线性变换：若 \(X \sim N(\mu, \sigma^2)\)，则 \(Y = aX + b \sim N(a\mu+b,\; a^2\sigma^2)\)，特别 \(\frac{X-\mu}{\sigma} \sim N(0,1)\)。

✎ 经典例题精讲

【例1 · 分布函数法】设 \(X \sim U(0,1)\)，求 \(Y = -2\ln X\) 的概率密度。

解：先求 \(Y\) 的分布函数 \(F_Y(y) = P(Y \le y) = P(-2\ln X \le y)\)
当 \(y > 0\) 时：\(P(-2\ln X \le y) = P(\ln X \ge -\frac{y}{2}) = P(X \ge e^{-y/2}) = 1 - e^{-y/2}\)
当 \(y \le 0\) 时：\(F_Y(y) = 0\)

求导得：\(f_Y(y) = \begin{cases} \frac{1}{2}e^{-y/2}, & y > 0 \\ 0, & y \le 0 \end{cases}\)
即 \(Y \sim Exp(\frac{1}{2})\)。
【例2 · 公式法】设 \(X \sim N(0,1)\)，\(Y = X^2\)，求 \(f_Y(y)\)。

解：当 \(y > 0\) 时，\(Y = x^2\) 的反函数为 \(x = \pm\sqrt{y}\)，有两个分支。
由公式法：\(f_Y(y) = f_X(\sqrt{y}) \cdot |(\sqrt{y})'| + f_X(-\sqrt{y}) \cdot |(-\sqrt{y})'|\)
\(= \frac{1}{\sqrt{2\pi}}e^{-y/2} \cdot \frac{1}{2\sqrt{y}} + \frac{1}{\sqrt{2\pi}}e^{-y/2} \cdot \frac{1}{2\sqrt{y}} = \frac{1}{\sqrt{2\pi y}} e^{-y/2}\)
当 \(y \le 0\) 时：\(f_Y(y) = 0\)。这正是 \(\chi^2(1)\) 分布的密度。
【例3 · 线性变换】设 \(X \sim N(2, 9)\)，求 \(Y = 2X + 1\) 的分布及 \(P(Y > 7)\)。

解：由正态分布的线性变换性质：\(Y = 2X + 1 \sim N(2 \times 2 + 1,\; 4 \times 9) = N(5, 36)\)
标准化：\(P(Y > 7) = P\left(\frac{Y-5}{6} > \frac{7-5}{6}\right) = P(Z > \frac{1}{3}) = 1 - \Phi(0.333) \approx 1 - 0.6304 = 0.3696\)

第三章 多维随机变量及其分布

▼

1联合分布与边缘分布

联合分布函数：\(F(x,y) = P(X \le x, Y \le y)\)

离散型——联合分布列与边缘分布列：

\(P(X = x_i) = \sum_j P(X = x_i, Y = y_j) = p_{i\cdot}\)

连续型——联合密度与边缘密度：

\(f_X(x) = \int_{-\infty}^{+\infty} f(x,y)\,dy, \quad f_Y(y) = \int_{-\infty}^{+\infty} f(x,y)\,dx\)

2独立性判断

X, Y 独立 \(\Leftrightarrow F(x,y) = F_X(x) \cdot F_Y(y)\)，对一切 \(x,y\) 成立

等价条件：

  • 离散型：\(p_{ij} = p_{i\cdot} \cdot p_{\cdot j}\) 对所有 \(i,j\) 成立
  • 连续型：\(f(x,y) = f_X(x) \cdot f_Y(y)\) 对几乎所有 \((x,y)\) 成立
若联合密度的定义域不能分解为两个独立变量各自的范围之积（如三角形区域），则 X, Y 一定不独立。

3条件分布

离散型：\(P(Y = y_j | X = x_i) = \frac{p_{ij}}{p_{i\cdot}}\)

连续型：\(f_{Y|X}(y|x) = \frac{f(x,y)}{f_X(x)},\; f_X(x) > 0\)

4二维正态分布

\((X,Y) \sim N(\mu_1, \mu_2; \sigma_1^2, \sigma_2^2; \rho)\)
  • 边缘分布：\(X \sim N(\mu_1, \sigma_1^2)\)，\(Y \sim N(\mu_2, \sigma_2^2)\)
  • \(\rho = 0\) \(\Leftrightarrow\) X 与 Y 独立 \(\Leftrightarrow\) X 与 Y 不相关（正态下三者等价）
  • 线性组合仍为正态：\(aX + bY \sim N(a\mu_1 + b\mu_2,\; a^2\sigma_1^2 + b^2\sigma_2^2 + 2ab\rho\sigma_1\sigma_2)\)

【例2 · 边缘密度与独立性】设 \((X,Y)\) 的联合密度为 \(f(x,y) = \begin{cases} cy(2-x), & 0 \le x \le 1,\; 0 \le y \le x \\ 0, & \text{其他} \end{cases}\) (1) 求常数 \(c\)；(2) 求边缘密度；(3) 判断 X, Y 是否独立。

解： (1) 由 \(\int\int f(x,y)\,dxdy = 1\)，积分区域为 \(0 \le y \le x \le 1\)： \(\int_0^1 \int_0^x cy(2-x)\,dy\,dx = \int_0^1 \frac{c}{2}x^2(2-x)\,dx = \frac{c}{2} \cdot \frac{5}{12} = 1\)，故 \(c = \frac{24}{5}\)

(2) \(f_X(x) = \int_0^x \frac{24}{5}y(2-x)\,dy = \frac{12}{5}x^2(2-x),\; 0 \le x \le 1\)
\(f_Y(y) = \int_y^1 \frac{24}{5}y(2-x)\,dx = \frac{12}{5}y(3-4y+y^2),\; 0 \le y \le 1\)

(3) 由于 \(f(x,y) \ne f_X(x) \cdot f_Y(y)\)，X 与 Y 不独立。
当联合密度的支撑区域不是矩形区域（如三角形、圆形），X 与 Y 一定不独立。
【例3 · 协方差计算】设 \((X,Y)\) 的联合分布律为
\(X \backslash Y\)-101
00.070.180.15
10.080.320.20
求 \(\text{Cov}(X,Y)\) 并判断 X, Y 是否独立。

解： \(E(X) = 0 \times 0.4 + 1 \times 0.6 = 0.6\) \(E(Y) = (-1) \times 0.15 + 0 \times 0.5 + 1 \times 0.35 = 0.2\) \(E(XY) = 0 \times 0 \times 0.18 + 1 \times (-1) \times 0.08 + 1 \times 1 \times 0.20 = 0.12\)
\(\text{Cov}(X,Y) = E(XY) - E(X)E(Y) = 0.12 - 0.6 \times 0.2 = 0\)
\(\text{Cov}(X,Y) = 0\)，X 与 Y 不相关。再验证独立性： \(p_{11} = 0.07\)，\(p_{1\cdot} \times p_{\cdot 1} = 0.4 \times 0.15 = 0.06 \ne 0.07\) 所以 X 与 Y 不独立，但不相关。这就是"不相关未必独立"的典型反例。

第四章 随机变量的数字特征

▼

1数学期望

离散型：\(E(X) = \sum_k x_k p_k\)；连续型：\(E(X) = \int_{-\infty}^{+\infty} x f(x)\,dx\)

随机变量函数的期望（LOTUS）：

\(E[g(X)] = \sum_k g(x_k) p_k\) 或 \(\int_{-\infty}^{+\infty} g(x) f(x)\,dx\)

性质：

  • \(E(c) = c\)，\(E(aX + b) = aE(X) + b\)
  • \(E(X + Y) = E(X) + E(Y)\)（不需要独立）
  • 若 X, Y 独立：\(E(XY) = E(X) \cdot E(Y)\)

2方差

\(D(X) = E[X - E(X)]^2 = E(X^2) - [E(X)]^2\)

性质：

  • \(D(c) = 0\)，\(D(aX + b) = a^2 D(X)\)
  • 若 X, Y 独立：\(D(X + Y) = D(X) + D(Y)\)
  • \(D(X \pm Y) = D(X) + D(Y) \pm 2\text{Cov}(X,Y)\)

3协方差与相关系数

协方差：\(\text{Cov}(X,Y) = E[(X - EX)(Y - EY)] = E(XY) - E(X)E(Y)\)
相关系数：\(\rho_{XY} = \dfrac{\text{Cov}(X,Y)}{\sqrt{D(X)}\,\sqrt{D(Y)}}\)

性质：

  • \(\text{Cov}(aX, bY) = ab\,\text{Cov}(X,Y)\)
  • \(\text{Cov}(X_1 + X_2, Y) = \text{Cov}(X_1, Y) + \text{Cov}(X_2, Y)\)
  • \(|\rho_{XY}| \le 1\)；\(|\rho_{XY}| = 1\) \(\Leftrightarrow\) 存在线性关系 \(P(Y = aX + b) = 1\)
  • \(\rho_{XY} = 0\) 称为 不相关
独立与不相关的关系：独立 \(\Rightarrow\) 不相关，反之不一定。但二维正态分布下：独立 \(\Leftrightarrow\) 不相关 \(\Leftrightarrow \rho = 0\)。

4矩与协方差矩阵

  • k 阶原点矩：\(E(X^k)\)
  • k 阶中心矩：\(E[X - E(X)]^k\)
  • \(E(X)\) 是一阶原点矩，\(D(X)\) 是二阶中心矩

其中 70 分钟 = 第一班 8:10 已过 + 等第二班 9:30，\(P(X=70) = \frac{1}{6} \times \frac{3}{6} = \frac{3}{36}\)
\(E(X) = 10 \times \frac{3}{6} + 30 \times \frac{2}{6} + 50 \times \frac{1}{36} + 70 \times \frac{3}{36} + 90 \times \frac{2}{36} = \frac{27.22}{1} \approx 27.22 \text{（分钟）}\)
【例2 · 期望技巧（指示变量法）】20 位旅客在 10 个车站下车（等可能），各旅客独立。若某站无人下车则不停车。求停车次数 \(X\) 的期望。

解：引入指示变量 \(X_i = \begin{cases} 0, & \text{第 } i \text{ 站无人下车} \\ 1, & \text{第 } i \text{ 站有人下车} \end{cases}\)，则 \(X = X_1 + X_2 + \cdots + X_{10}\)。
每位旅客不在第 \(i\) 站下车的概率 = \(\frac{9}{10}\)，20 人都不在第 \(i\) 站下车的概率 = \((\frac{9}{10})^{20}\)
\(E(X_i) = P(X_i = 1) = 1 - \left(\frac{9}{10}\right)^{20}\)
\(E(X) = 10 \times \left[1 - \left(\frac{9}{10}\right)^{20}\right] \approx 10 \times (1 - 0.1216) = 8.784\)
【例3 · 方差计算】设 \(X \sim U(a,b)\)，求 \(D(X)\)。

解：\(E(X) = \frac{a+b}{2}\)
\(E(X^2) = \int_a^b x^2 \cdot \frac{1}{b-a}\,dx = \frac{1}{3} \cdot \frac{b^3 - a^3}{b-a} = \frac{a^2 + ab + b^2}{3}\)
\(D(X) = E(X^2) - [E(X)]^2 = \frac{a^2+ab+b^2}{3} - \frac{(a+b)^2}{4} = \frac{(b-a)^2}{12}\)
【例4 · 分布可加性】设 \(X \sim \pi(2)\)，\(Y \sim \pi(2)\)，X 与 Y 独立，求 \(P(X+Y=3)\)。

解：由泊松分布可加性：\(X + Y \sim \pi(2+2) = \pi(4)\)
\(P(X+Y=3) = \frac{4^3 e^{-4}}{3!} = \frac{64 \times 0.0183}{6} \approx 0.1954\)
【例5 · 二项分布可加性】设 \(X \sim B(2, 0.5)\)，\(Y \sim B(3, 0.5)\)，X 与 Y 独立，求 \(P(X+Y=1)\)。

解：由二项分布可加性：\(X + Y \sim B(2+3, 0.5) = B(5, 0.5)\)
\(P(X+Y=1) = C_5^1 \times 0.5^1 \times 0.5^4 = 5 \times \frac{1}{32} = \frac{5}{32} = 0.15625\)

第五章 大数定律与中心极限定理

▼

1切比雪夫不等式

设 \(E(X)\) 和 \(D(X)\) 存在，则对任意 \(\varepsilon > 0\)： \(P\{|X - E(X)| \ge \varepsilon\} \le \dfrac{D(X)}{\varepsilon^2}\)
等价地：\(P\{|X - E(X)| < \varepsilon\} \ge 1 - \dfrac{D(X)}{\varepsilon^2}\)
切比雪夫不等式不需要知道分布形式，只需要期望和方差存在，可用于估计概率的上/下界。

2大数定律

  • 切比雪夫大数定律：若 \(X_1, X_2, \ldots\) 相互独立，且方差有上界 \(D(X_i) \le C\)，则 \(\frac{1}{n}\sum_{i=1}^n X_i \xrightarrow{P} \frac{1}{n}\sum_{i=1}^n E(X_i)\)
  • 伯努利大数定律：频率依概率收敛于概率：\(\frac{n_A}{n} \xrightarrow{P} p\)
  • 辛钦大数定律：若 \(X_i\) 独立同分布，\(E(X_i) = \mu\) 存在，则 \(\bar{X} \xrightarrow{P} \mu\)

3中心极限定理

独立同分布 CLT（林德伯格-列维）：
设 \(X_i\) iid，\(E(X_i) = \mu\)，\(D(X_i) = \sigma^2 > 0\)，则当 \(n\) 充分大时： \(\dfrac{\displaystyle\sum_{i=1}^n X_i - n\mu}{\sqrt{n}\,\sigma} \;\xrightarrow{d}\; N(0,1)\)
棣莫弗-拉普拉斯 CLT（二项近似正态）：
若 \(X \sim B(n,p)\)，则当 \(n\) 充分大时： \(\dfrac{X - np}{\sqrt{np(1-p)}} \;\xrightarrow{d}\; N(0,1)\)
【应用技巧】遇到"n 较大"的二项分布/独立求和问题时，用正态近似： \(P(a \le X \le b) \approx \Phi\!\left(\frac{b - np}{\sqrt{npq}}\right) - \Phi\!\left(\frac{a - np}{\sqrt{npq}}\right)\)

✎ 经典例题精讲

【例1 · 切比雪夫不等式】设 \(E(X) = \mu\)，\(D(X) = \sigma^2\)，用切比雪夫不等式估计 \(P(|X - \mu| \ge 3\sigma)\) 的上界。

解：取 \(\varepsilon = 3\sigma\)，由切比雪夫不等式： \(P(|X - \mu| \ge 3\sigma) \le \frac{D(X)}{(3\sigma)^2} = \frac{\sigma^2}{9\sigma^2} = \frac{1}{9}\)
即：随机变量偏离均值超过 3 倍标准差的概率不超过 \(\frac{1}{9} \approx 11.1\%\)。
【例2 · CLT 应用（二项近似正态）】某工厂产品的次品率为 0.05。求在 200 件产品中次品数不超过 15 件的概率。

解：设 X = 次品数，\(X \sim B(200, 0.05)\)。\(np = 10\)，\(\sqrt{np(1-p)} = \sqrt{9.5} \approx 3.082\)
由中心极限定理： \(P(X \le 15) = P\left(\frac{X - 10}{3.082} \le \frac{15 - 10}{3.082}\right) \approx \Phi(1.622) \approx 0.9474\)
【例3 · CLT 应用（独立求和）】一加法器同时收到 20 个噪声电压 \(V_k\)，各 \(V_k\) 独立同分布于 \(U(0,10)\)。求总电压 \(V = \sum_{k=1}^{20} V_k\) 超过 105 的概率。

解：每个 \(V_k\) 的 \(E(V_k) = 5\)，\(D(V_k) = \frac{100}{12}\)。
\(\sum V_k\) 的 \(E = 20 \times 5 = 100\)，\(D = 20 \times \frac{100}{12} = \frac{500}{3}\)
\(P(V > 105) = P\left(\frac{V - 100}{\sqrt{500/3}} > \frac{105 - 100}{\sqrt{500/3}}\right) \approx 1 - \Phi(0.387) \approx 1 - 0.6517 = 0.3483\)

三大抽样分布

▼

1\(\chi^2\) 分布

若 \(X_1, X_2, \ldots, X_n\) iid，\(X_i \sim N(0,1)\)，则 \(\chi^2 = X_1^2 + X_2^2 + \cdots + X_n^2 \sim \chi^2(n)\)

性质：

  • \(E(\chi^2) = n\)，\(D(\chi^2) = 2n\)
  • 可加性：若 \(X \sim \chi^2(n_1)\)，\(Y \sim \chi^2(n_2)\) 且独立，则 \(X + Y \sim \chi^2(n_1 + n_2)\)

2t 分布

若 \(X \sim N(0,1)\)，\(Y \sim \chi^2(n)\)，且 X, Y 独立，则 \(T = \dfrac{X}{\sqrt{Y/n}} \sim t(n)\)

性质：

  • t 分布关于 0 对称
  • 当 \(n\) 足够大时，\(t(n)\) 近似 \(N(0,1)\)
  • 若 \(T \sim t(n)\)，则 \(T^2 \sim F(1, n)\)

3F 分布

若 \(X \sim \chi^2(n_1)\)，\(Y \sim \chi^2(n_2)\)，且独立，则 \(F = \dfrac{X/n_1}{Y/n_2} \sim F(n_1, n_2)\)

性质：

  • 若 \(F \sim F(n_1, n_2)\)，则 \(\frac{1}{F} \sim F(n_2, n_1)\)
  • \(F_{1-\alpha}(n_1, n_2) = \frac{1}{F_\alpha(n_2, n_1)}\)

4正态总体下的抽样分布（重要！）

设 \(X_1, \ldots, X_n\) 是来自 \(N(\mu, \sigma^2)\) 的样本，\(\bar{X}\) 和 \(S^2\) 分别为样本均值和样本方差：

(1) \(\bar{X} \sim N\!\left(\mu, \dfrac{\sigma^2}{n}\right)\)，即 \(\dfrac{\bar{X} - \mu}{\sigma / \sqrt{n}} \sim N(0,1)\)
(2) \(\dfrac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)\)
(3) \(\bar{X}\) 与 \(S^2\) 相互独立
(4) \(\dfrac{\bar{X} - \mu}{S / \sqrt{n}} \sim t(n-1)\)
样本均值与样本方差的期望： \(E(\bar{X}) = \mu\)，\(D(\bar{X}) = \frac{\sigma^2}{n}\)，\(E(S^2) = \sigma^2\)

✎ 经典例题精讲

【例1 · 抽样分布性质】设总体 \(X \sim \pi(\lambda)\)（泊松分布），\(X_1, \ldots, X_n\) 是样本，求 \(E(\bar{X})\)，\(D(\bar{X})\) 和 \(E(S^2)\)。

解：泊松分布的 \(E(X) = \lambda\)，\(D(X) = \lambda\)。
\(E(\bar{X}) = E(X) = \lambda\)
\(D(\bar{X}) = \frac{D(X)}{n} = \frac{\lambda}{n}\)
\(E(S^2) = D(X) = \lambda\)
不论总体服从什么分布，\(\bar{X}\) 总是 \(\mu\) 的无偏估计，\(S^2\) 总是 \(\sigma^2\) 的无偏估计。
【例2 · 正态总体概率计算】设 \(X_1, \ldots, X_{25}\) 是来自 \(N(10, 4)\) 的样本，求 \(P(|\bar{X} - 10| > 0.5)\)。

解：\(\bar{X} \sim N(10, \frac{4}{25}) = N(10, 0.16)\)，即 \(\bar{X} \sim N(10, 0.4^2)\)。
\(P(|\bar{X} - 10| > 0.5) = 1 - P(|\bar{X} - 10| \le 0.5) = 1 - P\left(\left|\frac{\bar{X}-10}{0.4}\right| \le \frac{0.5}{0.4}\right)\)
\(= 1 - P(|Z| \le 1.25) = 1 - [2\Phi(1.25) - 1] = 2 - 2\Phi(1.25) = 2 - 2 \times 0.8944 = 0.2112\)
【例3 · t 分布构造】设 \(X_1, \ldots, X_{10}\) 是来自 \(N(0, \sigma^2)\) 的样本，求 \(\frac{\sqrt{10}\,\bar{X}}{S}\) 的分布。

解：\(\bar{X} \sim N(0, \frac{\sigma^2}{10})\)，故 \(\frac{\bar{X}}{\sigma/\sqrt{10}} \sim N(0,1)\)。
\(\frac{9S^2}{\sigma^2} \sim \chi^2(9)\)，且 \(\bar{X}\) 与 \(S^2\) 独立。
由 t 分布定义：\(\frac{\bar{X}/(\sigma/\sqrt{10})}{\sqrt{9S^2/(9\sigma^2)}} = \frac{\sqrt{10}\,\bar{X}}{S} \sim t(9)\)

第七章 参数估计

▼

1矩估计

基本思想：用样本矩 \(A_k = \frac{1}{n}\sum X_i^k\) 替换总体矩 \(E(X^k)\)，解出参数。

步骤：

  • 一个参数：令 \(E(X) = \bar{X}\)，解出 \(\hat{\theta}\)
  • 两个参数：令 \(E(X) = \bar{X}\)，\(E(X^2) = \frac{1}{n}\sum X_i^2\)，联立解出参数
【例】设 \(X \sim Exp(\lambda)\)，则 \(E(X) = 1/\lambda\)。令 \(1/\hat{\lambda} = \bar{X}\)，得 \(\hat{\lambda} = 1/\bar{X}\)。

2最大似然估计 (MLE)

基本思想：选取使样本出现概率最大的参数值。

步骤：

  • 写出似然函数 \(L(\theta) = \prod_{i=1}^n f(x_i; \theta)\)
  • 取对数 \(\ln L(\theta)\)
  • 对 \(\theta\) 求导令其为 0，解出 \(\hat{\theta}_{MLE}\)

另一种情形：若 \(L(\theta)\) 是单调增/减函数（如均匀分布），则通过观察确定最大值点（通常在边界取到）。

最大似然估计的不变性：若 \(\hat{\theta}\) 是 \(\theta\) 的 MLE，则 \(g(\hat{\theta})\) 是 \(g(\theta)\) 的 MLE。

3估计量的评价标准

  • 无偏性：\(E(\hat{\theta}) = \theta\)，则 \(\hat{\theta}\) 是 \(\theta\) 的无偏估计
  • 样本均值 \(\bar{X}\) 是 \(\mu\) 的无偏估计
  • 样本方差 \(S^2 = \frac{1}{n-1}\sum(X_i - \bar{X})^2\) 是 \(\sigma^2\) 的无偏估计（注意分母是 \(n-1\)！）
  • 有效性：在无偏估计中方差最小的更有效

4置信区间

置信水平 \(1-\alpha\) 下的置信区间表示参数有 \((1-\alpha)\times 100\%\) 的概率落在此区间内。

条件参数枢轴量置信区间
\(\sigma^2\) 已知 \(\mu\) \(Z = \frac{\bar{X}-\mu}{\sigma/\sqrt{n}}\) \(\bar{X} \pm z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}\)
\(\sigma^2\) 未知 \(\mu\) \(T = \frac{\bar{X}-\mu}{S/\sqrt{n}}\) \(\bar{X} \pm t_{\alpha/2}(n-1) \cdot \frac{S}{\sqrt{n}}\)
\(\mu\) 未知 \(\sigma^2\) \(\chi^2 = \frac{(n-1)S^2}{\sigma^2}\) \(\left[\frac{(n-1)S^2}{\chi^2_{\alpha/2}(n-1)},\; \frac{(n-1)S^2}{\chi^2_{1-\alpha/2}(n-1)}\right]\)

✎ 经典例题精讲

【例1 · 矩估计】设总体 \(X \sim U(a, b)\)，\(a, b\) 未知，\(X_1, \ldots, X_n\) 是样本，求 \(a, b\) 的矩估计量。

解： \(\mu_1 = E(X) = \frac{a+b}{2}\)，\(\mu_2 = E(X^2) = D(X) + [E(X)]^2 = \frac{(b-a)^2}{12} + \frac{(a+b)^2}{4}\)
联立方程：\(\begin{cases} a + b = 2\mu_1 \\ (b-a)^2 = 12(\mu_2 - \mu_1^2) \end{cases}\)
解得：\(\hat{a} = \bar{X} - \sqrt{3B_2}\)，\(\hat{b} = \bar{X} + \sqrt{3B_2}\)
其中 \(B_2 = \frac{1}{n}\sum(X_i - \bar{X})^2\) 为二阶中心矩。
【例2 · MLE（伯努利/二项）】设 \(X_1, \ldots, X_n\) 是来自 \(B(1, p)\) 的样本，求 \(p\) 的最大似然估计量。

解：\(P(X=x) = p^x(1-p)^{1-x},\; x=0,1\)
似然函数：\(L(p) = \prod_{i=1}^n p^{x_i}(1-p)^{1-x_i} = p^{\sum x_i}(1-p)^{n-\sum x_i}\)
取对数：\(\ln L = (\sum x_i)\ln p + (n - \sum x_i)\ln(1-p)\)
对 \(p\) 求导令其为 0：\(\frac{\sum x_i}{p} - \frac{n - \sum x_i}{1-p} = 0\)
解得：\(\hat{p}_{MLE} = \frac{1}{n}\sum X_i = \bar{X}\)
【例3 · MLE（正态总体）】设 \(X_1, \ldots, X_n\) 是来自 \(N(\mu, \sigma^2)\) 的样本，求 \(\mu, \sigma^2\) 的最大似然估计量。

解： \(\ln L = -\frac{n}{2}\ln(2\pi) - \frac{n}{2}\ln(\sigma^2) - \frac{1}{2\sigma^2}\sum(x_i - \mu)^2\)
对 \(\mu\) 求导：\(\frac{\partial \ln L}{\partial \mu} = \frac{1}{\sigma^2}\sum(x_i - \mu) = 0\) \(\Rightarrow\) \(\hat{\mu} = \bar{X}\)
对 \(\sigma^2\) 求导：\(\frac{\partial \ln L}{\partial(\sigma^2)} = -\frac{n}{2\sigma^2} + \frac{1}{2\sigma^4}\sum(x_i - \mu)^2 = 0\)
\(\hat{\sigma}^2 = \frac{1}{n}\sum(X_i - \bar{X})^2 = B_2\)
注意：MLE 求出的 \(\hat{\sigma}^2 = \frac{1}{n}\sum(X_i-\bar{X})^2\) 是有偏估计（分母为 n）。无偏估计量是样本方差 \(S^2 = \frac{1}{n-1}\sum(X_i-\bar{X})^2\)（分母为 n-1）。
【例4 · MLE（均匀分布，观察法）】设 \(X \sim U(0, \theta)\)，\(x_1, \ldots, x_n\) 是样本值，求 \(\theta\) 的 MLE。

解：似然函数 \(L(\theta) = \frac{1}{\theta^n}\)，约束条件 \(0 \le x_i \le \theta\)，即 \(\theta \ge x_{(n)} = \max(x_1, \ldots, x_n)\)。
\(L(\theta) = \frac{1}{\theta^n}\) 在 \(\theta \ge x_{(n)}\) 上是减函数，所以在 \(\theta = x_{(n)}\) 处取得最大值。
\(\hat{\theta}_{MLE} = X_{(n)} = \max(X_1, \ldots, X_n)\)
均匀分布的 MLE 不能用求导法，需通过观察似然函数的单调性确定。这是考试重点题型之一。
【例5 · 置信区间计算】某零件重量 \(N(\mu, \sigma^2)\)，\(\sigma^2\) 未知。抽取 16 个零件，测得 \(\bar{x} = 5.2\)（克），\(s = 0.4\)（克）。求 \(\mu\) 的 95% 置信区间。（\(t_{0.025}(15) = 2.1315\)）

解：\(\sigma^2\) 未知，使用 T 统计量：\(T = \frac{\bar{X} - \mu}{S/\sqrt{n}} \sim t(n-1)\)
置信区间：\(\bar{x} \pm t_{\alpha/2}(n-1) \cdot \frac{s}{\sqrt{n}} = 5.2 \pm 2.1315 \times \frac{0.4}{4}\)
\(= 5.2 \pm 0.21315 = [4.987,\; 5.413]\)

第八章 假设检验

▼

1基本步骤

  1. 提出假设：原假设 \(H_0\) 和备择假设 \(H_1\)
  2. 选择检验统计量
  3. 确定拒绝域：由显著性水平 \(\alpha\) 确定临界值
  4. 计算实测值
  5. 判断：实测值落入拒绝域则拒绝 \(H_0\)，否则接受 \(H_0\)
两类错误：
第一类（弃真）：\(H_0\) 为真却拒绝，概率为 \(\alpha\)
第二类（取伪）：\(H_0\) 为假却接受，概率为 \(\beta\)

2正态总体参数的假设检验

检验条件统计量双侧拒绝域
Z 检验 \(\sigma^2\) 已知，检验 \(\mu\) \(Z = \frac{\bar{X} - \mu_0}{\sigma/\sqrt{n}}\) \(|Z| \ge z_{\alpha/2}\)
T 检验 \(\sigma^2\) 未知，检验 \(\mu\) \(T = \frac{\bar{X} - \mu_0}{S/\sqrt{n}}\) \(|T| \ge t_{\alpha/2}(n-1)\)
\(\chi^2\) 检验 \(\mu\) 未知，检验 \(\sigma^2\) \(\chi^2 = \frac{(n-1)S^2}{\sigma_0^2}\) \(\chi^2 \ge \chi^2_{\alpha/2}\) 或 \(\le \chi^2_{1-\alpha/2}\)
【完整例题】某产品指标 \(N(\mu, 150^2)\)，\(\sigma = 150\) 已知。抽 16 个样本，均值 \(\bar{x} = 1607\)。检验 \(H_0: \mu = 1600\) vs \(H_1: \mu \ne 1600\)，\(\alpha = 0.05\)。

解： (1) \(H_0: \mu = 1600\)，\(H_1: \mu \ne 1600\)
(2) \(Z = \frac{\bar{X} - 1600}{150/\sqrt{16}} = \frac{1607 - 1600}{37.5} = 0.187\)
(3) 拒绝域：\(|Z| \ge z_{0.025} = 1.96\)
(4) \(|0.187| < 1.96\)，未落入拒绝域
(5) 接受 \(H_0\)，可以认为该批产品指标均值为 1600。
【T 检验例题】某厂灯泡寿命 \(N(\mu, \sigma^2)\)，\(\sigma^2\) 未知。抽取 9 只，\(\bar{x} = 1120\) 小时，\(s = 30\) 小时。检验 \(H_0: \mu = 1100\) vs \(H_1: \mu > 1100\)，\(\alpha = 0.05\)。（\(t_{0.05}(8) = 1.8595\)）

解： (1) \(H_0: \mu = 1100\)，\(H_1: \mu > 1100\)（右侧检验）
(2) \(T = \frac{\bar{X} - 1100}{S/\sqrt{n}} = \frac{1120 - 1100}{30/\sqrt{9}} = \frac{20}{10} = 2.0\)
(3) 拒绝域：\(T \ge t_{0.05}(8) = 1.8595\)
(4) \(2.0 > 1.8595\)，落入拒绝域
(5) 拒绝 \(H_0\)，认为灯泡平均寿命显著大于 1100 小时。
【\(\chi^2\) 检验例题】纤维强力 \(N(\mu, \sigma^2)\)，\(\mu\) 未知。抽取 20 根，\(s^2 = 2.5\)。检验 \(H_0: \sigma^2 = 4\) vs \(H_1: \sigma^2 < 4\)，\(\alpha = 0.05\)。（\(\chi^2_{0.95}(19) = 10.117\)）

解： (1) \(H_0: \sigma^2 = 4\)，\(H_1: \sigma^2 < 4\)（左侧检验）
(2) \(\chi^2 = \frac{(n-1)S^2}{\sigma_0^2} = \frac{19 \times 2.5}{4} = 11.875\)
(3) 拒绝域：\(\chi^2 \le \chi^2_{0.95}(19) = 10.117\)
(4) \(11.875 > 10.117\)，未落入拒绝域
(5) 接受 \(H_0\)，不能认为方差有显著减小。

3单侧检验

当关心参数是否偏大或偏小时，使用单侧检验：

  • 右侧检验：\(H_0: \mu \le \mu_0\) vs \(H_1: \mu > \mu_0\)，拒绝域：\(Z \ge z_\alpha\)
  • 左侧检验：\(H_0: \mu \ge \mu_0\) vs \(H_1: \mu < \mu_0\)，拒绝域：\(Z \le -z_\alpha\)

概率公式

加法公式：\(P(A \cup B) = P(A) + P(B) - P(AB)\)
条件概率：\(P(B|A) = \frac{P(AB)}{P(A)}\)
全概率：\(P(B) = \sum_{i=1}^n P(A_i)P(B|A_i)\)
贝叶斯：\(P(A_i|B) = \frac{P(A_i)P(B|A_i)}{\sum_j P(A_j)P(B|A_j)}\)

数字特征公式

\(D(X) = E(X^2) - [E(X)]^2\)
\(\text{Cov}(X,Y) = E(XY) - E(X)E(Y)\)
\(D(X \pm Y) = D(X) + D(Y) \pm 2\text{Cov}(X,Y)\)
\(\rho_{XY} = \frac{\text{Cov}(X,Y)}{\sqrt{D(X)}\sqrt{D(Y)}}\)

统计量公式

样本均值：\(\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i\)
样本方差：\(S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \bar{X})^2\)
正态下：\(\bar{X} \sim N(\mu, \sigma^2/n)\)，\(\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)\)，\(\frac{\bar{X}-\mu}{S/\sqrt{n}} \sim t(n-1)\)

概率论易错点

  • 互斥 vs 独立：互斥（\(AB = \varnothing\)）不等于独立。互斥事件若概率都 > 0，则一定不独立。
  • 条件概率陷阱：\(P(A|B) \ne P(B|A)\)，贝叶斯公式正是用来区分这两者。
  • 分布函数法做题时：\(F_Y(y) = P(Y \le y)\) 要注意 \(Y = g(X)\) 的取值范围，分段讨论。
  • 公式法的条件：\(y = g(x)\) 必须严格单调。不单调时只能用分布函数法。
  • 边缘分布不能确定联合分布：知道 \(f_X(x)\) 和 \(f_Y(y)\) 不能唯一确定 \(f(x,y)\)，还需要知道相关结构。
  • 不相关 ≠ 独立：\(\rho = 0\)（不相关）不代表独立。只有二维正态分布下两者等价。

数理统计易错点

  • 样本方差分母是 n-1：\(S^2 = \frac{1}{n-1}\sum(X_i - \bar{X})^2\)，不是 \(n\)！用 n 做分母的是 \(B_2 = \frac{1}{n}\sum(X_i - \bar{X})^2\)。
  • \(\bar{X}\) 与 \(S^2\) 独立只在正态总体下成立，一般总体不保证。
  • \(\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)\)，自由度是 \(n-1\) 不是 \(n\)。
  • \(\frac{\bar{X}-\mu}{S/\sqrt{n}} \sim t(n-1)\)，自由度也是 \(n-1\)。
  • MLE 的不变性：若 \(\hat{\theta}\) 是 MLE，则 \(g(\hat{\theta})\) 是 \(g(\theta)\) 的 MLE。但矩估计没有此性质。
  • 均匀分布 MLE：不能用求导法，需观察似然函数单调性。\(\hat{\theta} = X_{(n)} = \max(X_1, \ldots, X_n)\)。
  • 假设检验：先确定是双侧还是单侧，再选择对应的拒绝域。不要混淆 \(z_{\alpha}\) 和 \(z_{\alpha/2}\)。

计算技巧

  • 正态分布标准化：先减 \(\mu\) 再除 \(\sigma\)，\(\frac{X-\mu}{\sigma} \sim N(0,1)\)
  • 泊松分布：\(E(X) = D(X) = \lambda\)，期望等于方差
  • 利用分布可加性简化计算，比直接计算联合分布快得多
  • 指示变量法求期望：将复杂随机变量拆成 0-1 变量之和
  • 检验统计量与枢轴量的区别：检验统计量用 \(\mu_0\)（假设值），枢轴量用 \(\mu\)（真实参数）
