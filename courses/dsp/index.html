第1章 离散时间信号

1.1 基本概念与典型序列

离散时间信号 $x(n)$ 是从连续时间信号 $x_a(t)$ 通过采样得到的：将连续变量 $t$ 替换为 $nT$（$T$ 为采样间隔/采样周期）。采样频率 $F_s = 1/T$。

六种典型序列

序列定义说明
单位脉冲序列$\delta(n) = \begin{cases}1, & n=0\\0, & n\neq 0\end{cases}$注意与 $\delta(t)$ 的区别
单位阶跃序列$u(n) = \begin{cases}1, & n\geq 0\\0, & n<0\end{cases}$
矩形序列$R_N(n) = \begin{cases}1, & 0\leq n\leq N-1\\0, & \text{其他}\end{cases}$$N$ 为非零区间长度
实指数序列$x(n) = a^n u(n)$$|a|<1$ 收敛，$|a|>1$ 发散
正弦序列$x(n) = A\sin(\omega_0 n + \varphi)$$\omega_0$ 为数字角频率
复指数序列$x(n) = e^{(\sigma+j\omega_0)n}$实部为余弦，虚部为正弦

1.2 重要公式与性质

核心公式
数字角频率与模拟角频率的关系：
$$\boxed{\omega = \Omega T = \frac{2\pi f}{F_s}}$$ 其中 $\omega$ 为数字角频率，$\Omega$ 为模拟角频率，$f$ 为物理频率，$F_s$ 为采样频率。
数字角频率 $\omega$ 是物理频率 $f$ 对采样频率 $F_s$ 的归一化值（相对值），这个公式必须记住！

正弦序列的周期性判断

当 $\frac{2\pi}{\omega_0} = \frac{N}{i}$（$i$ 为整数）时，正弦序列是周期序列，周期为 $N$。
- 若 $\frac{2\pi}{\omega_0}$ 为有理数：是周期序列
- 若 $\frac{2\pi}{\omega_0}$ 为无理数：不是周期序列

例题：判断正弦序列周期性（点击展开）

判断 $\cos\left(\frac{3\pi}{7}n\right)$ 是否为周期序列。 Step 1｜提取数字角频率 $$\omega_0 = \frac{3\pi}{7}$$ Step 2｜计算 $\frac{2\pi}{\omega_0}$ $$\frac{2\pi}{\omega_0} = \frac{2\pi}{3\pi/7} = \frac{14}{3}$$ Step 3｜判断是否为有理数
$\frac{14}{3}$ 是有理数 → 是周期序列 Step 4｜求最小周期
$\frac{2\pi}{\omega_0} = \frac{N}{i} = \frac{14}{3}$，取 $N=14, i=3$（最简分数） $$\boxed{N = 14}$$ 验证：$\cos\left(\frac{3\pi}{7}(n+14)\right) = \cos\left(\frac{3\pi}{7}n + 6\pi\right) = \cos\left(\frac{3\pi}{7}n\right)$ ✓
两个周期序列之和的周期 = 各序列周期的最小公倍数。

序列的基本运算

运算公式
偶部$x_e(n) = \frac{1}{2}[x(n) + x(-n)]$
奇部$x_o(n) = \frac{1}{2}[x(n) - x(-n)]$
单位脉冲表示任意序列：$x(n) = \sum_{k=-\infty}^{\infty} x(k)\delta(n-k)$
线性卷积$y(n) = x(n)*h(n) = \sum_{k=-\infty}^{\infty} x(k)h(n-k)$
线性卷积的不进位乘法：右端对齐，每列相乘后求和（不进位），输出序列的起止点 = 两个输入序列起止点分别相加。

变换域分析

DTFT 离散时间傅里叶变换

DTFT 正变换
$$X(e^{j\omega}) = \sum_{n=-\infty}^{\infty} x(n)e^{-j\omega n}$$
IDTFT 反变换
$$x(n) = \frac{1}{2\pi}\int_{-\pi}^{\pi} X(e^{j\omega})e^{j\omega n}d\omega$$
这个公式在第5章 FIR 滤波器设计中会用到，用于求理想滤波器的单位脉冲响应！
DTFT 频谱 $X(e^{j\omega})$ 是以 $2\pi$ 为周期的连续周期频谱。存在条件：Z 变换包含单位圆。

DTFT 重要性质

性质时域频域
线性$ax_1(n)+bx_2(n)$$aX_1(e^{j\omega})+bX_2(e^{j\omega})$
时移$x(n-n_0)$$e^{-j\omega n_0}X(e^{j\omega})$
频移$e^{j\omega_0 n}x(n)$$X(e^{j(\omega-\omega_0)})$
共轭$x^*(n)$$X^*(e^{-j\omega})$
实偶序列$x(n)$ 实偶$X(e^{j\omega})$ 实偶
实奇序列$x(n)$ 实奇$X(e^{j\omega})$ 虚奇
实序列$x(n)$ 实$X(e^{j\omega})$ 共轭偶对称
对称性速记：实偶→频谱实偶，实奇→频谱虚奇，实序列→频谱共轭偶对称。

对称性示意对比图

① 实偶序列 → 频谱 实偶

x(n)关于n=0对称 ↔ X(e^jω)为纯实数且关于ω=0对称（虚部为0）

② 实奇序列 → 频谱 虚奇

x(n)关于n=0反对称且x(0)=0 ↔ X(e^jω)为纯虚数且关于ω=0反对称（实部为0）

③ 一般实序列 → 频谱 共轭偶对称

$X(e^{j\omega}) = X^*(e^{-j\omega})$：幅度谱偶对称，相位谱奇对称 → 幅度信息无损，只需分析 $[0,\pi]$ 即可

例题：求 $x(n) = a^n u(n)$ 的 DTFT（点击展开）

已知：$x(n) = a^n u(n)$，$|a|<1$，求 $X(e^{j\omega})$ 和幅度谱。 Step 1｜写出 DTFT 定义式 $$X(e^{j\omega}) = \sum_{n=-\infty}^{\infty} x(n)e^{-j\omega n}$$ Step 2｜代入序列，确定求和范围
因为 $u(n) = 0$（$n<0$），所以求和下限从 $-\infty$ 变为 $0$： $$X(e^{j\omega}) = \sum_{n=0}^{\infty} a^n e^{-j\omega n} = \sum_{n=0}^{\infty} (ae^{-j\omega})^n$$ Step 3｜等比级数求和
这是一个公比为 $r = ae^{-j\omega}$ 的无穷等比级数，当 $|r| = |a| < 1$ 时收敛： $$X(e^{j\omega}) = \frac{1}{1 - ae^{-j\omega}}, \quad |a|<1$$ Step 4｜求幅度谱
将分母写成实部 + 虚部形式： $$1 - ae^{-j\omega} = 1 - a\cos\omega + ja\sin\omega$$ $$|1 - ae^{-j\omega}|^2 = (1-a\cos\omega)^2 + (a\sin\omega)^2 = 1 - 2a\cos\omega + a^2$$ $$\boxed{|X(e^{j\omega})| = \frac{1}{\sqrt{1 - 2a\cos\omega + a^2}}}$$ Step 5｜关键观察
- 当 $\omega = 0$ 时，$|X| = \frac{1}{|1-a|}$（最大值）
- 当 $\omega = \pi$ 时，$|X| = \frac{1}{|1+a|}$（最小值）
- 幅度谱关于 $\omega = 0$ 偶对称（因为 $x(n)$ 是实序列）

Z 变换

Z 变换定义
$$X(z) = \sum_{n=-\infty}^{\infty} x(n)z^{-n}$$
收敛域 (ROC) 是以原点为边界的环形区域。内径可缩到零点（实心圆），外径可扩展到无穷（空心圆）。

常用 Z 变换对（必须记住！）

序列 $x(n)$$X(z)$收敛域
$\delta(n)$$1$全 z 平面
$a^n u(n)$（因果等比）$\frac{1}{1-az^{-1}}$$|z|>|a|$（圆外）
$-a^n u(-n-1)$（反因果）$\frac{1}{1-az^{-1}}$$|z|<|a|$（圆内）
同一个 Z 变换表达式对应不同收敛域 → 不同的序列！求逆变换时必须根据收敛域判断是因果还是反因果序列。
求零极点的常见错误：当 $H(z)$ 是 $z^{-1}$ 的有理式时，一定要分子分母同乘 $z^M$ 化为正幂次多项式再求零极点，否则会遗漏！

零极点分布图

零点：空心小圆圈 ○，标注值；极点：×，标注值。
收敛域：以原点为边界的环形区域，用阴影或文字标注。
因果序列收敛域 = 极点最大模为半径的圆外；包含无穷点。

Z 变换的性质

性质说明
线性$a x_1(n) + b x_2(n) \leftrightarrow aX_1(z)+bX_2(z)$
卷积$x(n)*h(n) \leftrightarrow X(z)\cdot H(z)$

★ 时移（位移性）详解

时域视角：序列的移位

对序列 $x(n)$ 进行时域移位 $n_0$ 个样本：

  • $x(n-n_0)$：序列右移 $n_0$（延迟）
  • $x(n+n_0)$：序列左移 $n_0$（超前）
时域移位 → Z 域
$$x(n-n_0) \leftrightarrow z^{-n_0}X(z)$$
物理意义：时域每延迟一个样本，在 Z 域就多乘一个 $z^{-1}$。
$z^{-1}$ 就是一个单位延迟器。
频域视角：相位的旋转

在单位圆上取值（$z = e^{j\omega}$），时移性质变为：

频域效果（DTFT）
$$x(n-n_0) \leftrightarrow e^{-j\omega n_0} X(e^{j\omega})$$
关键结论：
- 时域移位不改变幅度谱：$|e^{-j\omega n_0} X(e^{j\omega})| = |X(e^{j\omega})|$
- 时域移位只改变相位谱：相位增加 $-\omega n_0$（线性相移）
- 这就是线性相位 FIR 滤波器的理论基础！
逆 Z 变换：掌握部分分式展开法。两种求待定系数的方法：①遮挡法 ②通分对应系数相等法。根据收敛域选择因果/反因果对应公式。

离散时间系统

系统特性的判别

线性时不变（LTI）系统经验判别法

线性判别
系统差分方程中只能出现：
  1. $x(n)$、$y(n)$ 及其一次项（不能有平方、对数等非线性运算）
  2. 系数不能包含 $n$
时不变判别
  1. $x(n)$、$y(n)$ 及其移位项前面的系数不能包含 $n$
  2. $x(n)$、$y(n)$ 前面的系数必须为 1

稳定性与因果性

稳定条件
$$\sum_{n=-\infty}^{\infty} |h(n)| < \infty \quad \text{（绝对可和）}$$
因果条件
$$h(n) = 0, \quad n < 0$$ 即 $h(n)$ 是右边序列。
从零极点/收敛域判断：
- 因果：收敛域包含无穷（圆外）
- 稳定：收敛域包含单位圆
- 既因果又稳定：所有极点在单位圆内

第2章 信号的采样与重建

采样定理

奈奎斯特采样定理
要使带限信号采样后能够不失真还原，采样频率必须满足：
$$\boxed{F_s \geq 2f_{\max}}$$ 其中 $f_{\max}$ 是信号的最高频率。$2f_{\max}$ 称为奈奎斯特率。

数字信号处理系统框图

输入 → [抗混叠滤波器] → [A/D转换器] → [数字信号处理器] → [D/A转换器] → [平滑滤波器] → 输出
         模拟低通          采样保持           计算机           数字转模拟         去除高频镜像
         防止混叠           连续→离散          信号处理          离散→连续          平滑输出
各部分作用要了解：
- 抗混叠滤波器：模拟低通，滤除高于 $F_s/2$ 的频率分量
- A/D：采样 + 量化，连续信号 → 数字信号
- D/A：数字信号 → 模拟信号
- 平滑滤波器：去除 D/A 输出中的高频镜像分量

第3章 DFT 与 FFT

3.1 DFT 离散傅里叶变换

DFS（离散傅里叶级数）不考，只需掌握 DFT 和 FFT。
DFT 正变换
$$X(k) = \sum_{n=0}^{N-1} x(n) W_N^{kn}, \quad k=0,1,\ldots,N-1$$
IDFT 反变换
$$x(n) = \frac{1}{N}\sum_{k=0}^{N-1} X(k) W_N^{-kn}, \quad n=0,1,\ldots,N-1$$
旋转因子
$$W_N = e^{-j\frac{2\pi}{N}}, \quad W_N^{kn} = e^{-j\frac{2\pi}{N}kn}$$ $W_N$ 以 $N$ 为周期。
DFT 的时域和频域都是有限长离散序列，是计算机中使用最多的变换。

矩阵运算法计算 DFT

$$\mathbf{X} = \mathbf{W} \cdot \mathbf{x}$$ W 矩阵的构造：行号从 0 开始，列号从 0 开始，每个元素 $W_N^{kn}$（行号×列号），下标始终为 $N$。
考试时矩阵乘法的步骤不能省略，否则扣分！

DFT 的对称性

与 DTFT 的对称性结论一致：
- 实偶序列 → DFT 频谱实偶
- 实奇序列 → DFT 频谱虚奇
- 实序列 → DFT 共轭偶对称
区别：DTFT 是连续的，DFT 是离散的。

3.2 循环卷积

循环卷积的列表法（必须掌握！）

四点循环卷积步骤：
  1. 补零：两个序列都补零到长度 $L$（如 4 点）
  2. 折叠：将 $x_2(m)$ 折叠得到 $x_2(-m)$——位置 0 不变，其余从尾部往前读
  3. 相乘求和：$x_2(-m)$ 与 $x_1(m)$ 对应项相乘求和，得到第一个值
  4. 循环右移：将 $x_2(-m)$ 右移一位（尾部移回头部），再与 $x_1(m)$ 对应项相乘求和
  5. 重复步骤 4，得到所有 $L$ 个值
循环卷积与线性卷积的关系
当循环卷积长度 $L \geq N_1 + N_2 - 1$（$N_1, N_2$ 为两序列非零长度）时，
循环卷积可以完全代替线性卷积。

3.3 三大变换之间的关系

关系公式/描述
Z 变换 → DTFT$X(e^{j\omega}) = X(z)\big|_{z=e^{j\omega}}$
单位圆上的 Z 变换即 DTFT

3.4 帕塞瓦尔定理

物理意义：时域中对序列求能量 = 频域中求序列能量，能量守恒。
公式不用记，只要知道物理概念即可。

FFT 快速傅里叶变换

FFT 基本概念与流图

FFT 不是新变换，是 DFT 的快速算法。基本思想：利用 $W_N$ 系数的周期性和对称性，将大点数 DFT 分解为小点数 DFT，减少运算量。

蝶形运算基本单元

蝶形运算单元：W 系数在输入端下路，上路输出 = 两输入相加，下路输出 = 两输入相减后再乘 W 系数

★ 按时间抽取的 N=8 点 FFT 完整流图

图 3.19：按时间抽取的 N=8 点 FFT 运算流图（重点！必须会画）

画流图关键记忆点：
  • 8 点 = 3 级，$N$ 点 = $\log_2 N$ 级
  • 蝶形基本形状：上路输出 = 相加，下路输出 = 相减（W 系数在下路输入端）
  • 各级蝶距（间隔）：第1级 = 1，第2级 = 2，第3级 = 4，…，第 $M$ 级 = $N/2$
  • 输入序列：码位倒序（0, 4, 2, 6, 1, 5, 3, 7）——二进制位反转
  • 输出序列：自然顺序（0, 1, 2, 3, 4, 5, 6, 7）
  • 第 1 级只有一种 $W_N^0$；中间级上下对称；最后一级 $W$ 系数是等比序列 $W_N^0, W_N^1, \ldots$
  • 支持原位计算：每一级运算结果覆盖原存储器，无需额外空间

基2 FFT：复乘 $\frac{N}{2}\log_2 N$，复加 $N\log_2 N$

例题：N=256 时 FFT 效率对比（点击展开）

已知 $N=256$，对比直接 DFT 与 FFT 的运算量。 Step 1｜直接 DFT 运算量
复乘次数：$N^2 = 256^2 = 65536$
复加次数：$N(N-1) = 256 \times 255 = 65280$ Step 2｜基2 FFT 运算量
复乘次数：$\frac{N}{2}\log_2 N = \frac{256}{2} \times \log_2 256 = 128 \times 8 = 1024$
复加次数：$N\log_2 N = 256 \times 8 = 2048$ Step 3｜效率对比
$$\text{复乘效率提升} = \frac{N^2}{\frac{N}{2}\log_2 N} = \frac{2N}{\log_2 N} = \frac{2 \times 256}{8} = \boxed{64 \text{ 倍}}$$ 复加效率提升：$\frac{65280}{2048} \approx 32$ 倍 总结：$N$ 越大，FFT 优势越明显。$N=1024$ 时提升约 200 倍。

FFT 应用

快速卷积

快速卷积四个步骤（必须记住）：
  1. 对两个序列做 $L$ 点 DFT（$L \geq N_1+N_2-1$）
  2. 频域相乘：$Y(k) = X(k) \cdot H(k)$
  3. 对 $Y(k)$ 做 IDFT
  4. 当 DFT 换成 FFT 时，即为快速卷积

4.1 模拟滤波器原型（概念题）

模拟滤波器不考大计算题，只考概念。

三种模拟滤波器对比

滤波器幅度特性阶次参数灵敏度
巴特沃斯通带阻带都单调下降，通带内最大平坦最高最低（最不灵敏/最佳）
切比雪夫通带和阻带一个等波纹一个单调中等中等

滤波器阶次越高，过渡带越窄。

4.2 脉冲响应不变法

映射关系
$$z = e^{sT} \quad \Rightarrow \quad \omega = \Omega T$$ 数字角频率 = 模拟角频率 × 采样间隔（线性关系）
步骤：将模拟滤波器 $H_a(s)$ 做部分分式展开，每个极点 $s_i$ 对应 $z$ 域的 $e^{s_i T}$。
优点：频率变换是线性关系
缺点：有频谱混叠效应（模拟频谱的周期延拓导致）
适用：只能设计低通和带通滤波器，不能设计高通和带阻！

★ 例题：脉冲响应不变法设计一阶低通（点击展开详细步骤）

题目：已知 $F_s = 1000$ Hz，用脉冲响应不变法设计一个 3dB 截止频率 $f_c = 200$ Hz 的一阶巴特沃斯低通数字滤波器。求 $H(z)$。 Step 1｜求数字截止频率 $$\omega_c = 2\pi \frac{f_c}{F_s} = 2\pi \times \frac{200}{1000} = 0.4\pi$$ Step 2｜求模拟截止频率（线性映射 $\Omega = \omega / T$） $$\Omega_c = \frac{\omega_c}{T} = \omega_c \cdot F_s = 0.4\pi \times 1000 = 400\pi \approx 1256.64 \text{ rad/s}$$ Step 3｜写出模拟滤波器传递函数
一阶巴特沃斯归一化原型：$H_a(p) = \frac{1}{p+1}$
反归一化（$p \to s/\Omega_c$）： $$H_a(s) = \frac{\Omega_c}{s + \Omega_c} = \frac{1256.64}{s + 1256.64}$$ Step 4｜部分分式展开
极点：$s_1 = -\Omega_c = -1256.64$，展开结果与 $H_a(s)$ 本身一致（只有一个极点）。 Step 5｜脉冲响应不变法：极点映射
规则：每个极点 $s_i$ 映射到 $z$ 域 $e^{s_i T}$，系数乘以 $T$： $$H(z) = T \cdot \sum_i \frac{A_i}{1 - e^{s_i T} z^{-1}}$$ Step 6｜代入数值 $$\Omega_c T = 0.4\pi \approx 1.2566$$ $$e^{-\Omega_c T} = e^{-1.2566} \approx 0.2846$$ $$T \cdot \Omega_c = 0.001 \times 1256.64 = 1.2566$$ $$\boxed{H(z) = \frac{1.2566}{1 - 0.2846\,z^{-1}}}$$ Step 7｜验证
- $z=1$（$\omega=0$）：$H(1) = \frac{1.2566}{0.7154} \approx 1.757$（低频增益大）
- $z=-1$（$\omega=\pi$）：$H(-1) = \frac{1.2566}{1.2846} \approx 0.978$（高频增益小）
- 低频 > 高频 ✓，符合低通特性

4.3 双线性变换法

变换公式（必须记住！）
$$\boxed{s = \frac{2}{T} \cdot \frac{1-z^{-1}}{1+z^{-1}}}$$
频率映射关系
$$\boxed{\Omega = \frac{2}{T}\tan\frac{\omega}{2}}$$ 模拟角频率与数字角频率是非线性关系。
优点：无频谱混叠（S 平面到 Z 平面一一对应）
缺点：频率映射是非线性关系（会产生频率畸变）

★ 例题：双线性变换法设计二阶低通（点击展开详细步骤）

题目：已知 $F_s = 8000$ Hz，用双线性变换法设计一个 3dB 截止频率 $f_c = 1000$ Hz 的二阶巴特沃斯低通数字滤波器。求 $H(z)$。 Step 1｜求数字截止频率 $$\omega_c = 2\pi \frac{f_c}{F_s} = 2\pi \times \frac{1000}{8000} = \frac{\pi}{4}$$ Step 2｜频率预畸变（双线性变换的核心步骤！） $$\Omega_c = \frac{2}{T}\tan\frac{\omega_c}{2} = \frac{2}{T}\tan\frac{\pi}{8}$$ 这里 $T$ 取任意值（通常最后会约掉），令 $T=1$： $$\Omega_c = 2\tan\frac{\pi}{8} = 2(\sqrt{2}-1) \approx 0.8284$$ Step 3｜二阶巴特沃斯归一化原型 $$H_a(p) = \frac{1}{p^2 + \sqrt{2}\,p + 1}$$ Step 4｜反归一化（$p \to s/\Omega_c$） $$H_a(s) = \frac{\Omega_c^2}{s^2 + \sqrt{2}\,\Omega_c\,s + \Omega_c^2}$$ 其中 $\Omega_c^2 = 4(\sqrt{2}-1)^2 = 12 - 8\sqrt{2} \approx 0.6863$ Step 5｜双线性变换代入
将 $s = 2\cdot\frac{1-z^{-1}}{1+z^{-1}}$ 代入 $H_a(s)$： 计算各项： $$s^2 = 4\left(\frac{1-z^{-1}}{1+z^{-1}}\right)^2 = \frac{4(1-z^{-1})^2}{(1+z^{-1})^2} = \frac{4 - 8z^{-1} + 4z^{-2}}{(1+z^{-1})^2}$$ $$\sqrt{2}\,\Omega_c \cdot s = \frac{2\sqrt{2}\,\Omega_c(1-z^{-1})}{1+z^{-1}} = \frac{2\sqrt{2}\,\Omega_c(1+z^{-1})(1-z^{-1})}{(1+z^{-1})^2} = \frac{2\sqrt{2}\,\Omega_c(1-z^{-2})}{(1+z^{-1})^2}$$ 代入并通分（分母共同为 $(1+z^{-1})^2$）： Step 6｜整理分子 $$\text{分子} = \Omega_c^2 = 12 - 8\sqrt{2} \approx 0.6863$$ Step 7｜整理分母
常数项：$4 + 2\sqrt{2}\,\Omega_c + \Omega_c^2 = 4 + 2\sqrt{2}\cdot 2(\sqrt{2}-1) + (12-8\sqrt{2})$
$= 4 + (8-4\sqrt{2}) + 12 - 8\sqrt{2} = 24 - 12\sqrt{2} \approx 7.029$ $z^{-1}$ 项系数：$-8 + 0 = -8$ $z^{-2}$ 项系数：$4 - 2\sqrt{2}\,\Omega_c + \Omega_c^2 = 4 - (8-4\sqrt{2}) + 12 - 8\sqrt{2}$
$= 8 - 4\sqrt{2} \approx 2.343$ $$H(z) = \frac{0.6863}{(7.029 - 8z^{-1} + 2.343z^{-2})(1+z^{-1})^2/(1+z^{-1})^2}$$ 化简后： $$H(z) = \frac{0.6863(1+z^{-1})^2}{7.029 - 8z^{-1} + 2.343z^{-2}}$$ 展开分子：$0.6863 + 1.3726z^{-1} + 0.6863z^{-2}$ Step 8｜归一化（分母常数项化为 1）
各项除以 $7.029$： $$\boxed{H(z) = \frac{0.0976 + 0.1953\,z^{-1} + 0.0976\,z^{-2}}{1 - 1.1382\,z^{-1} + 0.3334\,z^{-2}}}$$ Step 9｜验证
- $z=1$（$\omega=0$）：$H(1) = \frac{0.0976 \times 4}{1 - 1.1382 + 0.3334} = \frac{0.3905}{0.1952} \approx 2.0$
- $z=-1$（$\omega=\pi$）：$H(-1) = \frac{0}{1+1.1382+0.3334} = 0$（二阶巴特沃斯在 $\omega=\pi$ 处有二阶零点）
- 低频增益大，高频衰减到 0 ✓ ★ 做题提醒：
  1. 频率预畸变 $\Omega_c = \frac{2}{T}\tan\frac{\omega_c}{2}$ 不能省略！
  2. $T$ 通常在代入过程中约掉，考试时可以先不代入具体值
  3. 考试一般考二阶
做题技巧：不要急于代入 $T$ 和 $\Omega_c$ 的具体值！先保留符号，很多项会约掉，计算更简单。考试一般考二阶。

第5章 FIR 滤波器设计

5.1 FIR 滤波器特点

FIR 的三大特点：
  1. 永远稳定：极点全部在原点
  2. 总是因果物理可实现的
  3. 满足一定对称性条件下，可得到严格的线性相位
"FIR 滤波器都具有线性相位" 这句话是错误的！必须满足 $h(n)$ 是实序列且满足奇/偶对称条件。

5.2 线性相位 FIR 滤波器

四类线性相位 FIR 滤波器

类型对称性N相位函数
第1类偶对称奇数$-\omega\frac{N-1}{2}$
第2类偶对称偶数$-\omega\frac{N-1}{2}$
第3类奇对称奇数$-\omega\frac{N-1}{2}-\frac{\pi}{2}$
第4类奇对称偶数$-\omega\frac{N-1}{2}-\frac{\pi}{2}$
记忆：先记第1类（偶对称+奇数），然后按 "偶偶→偶奇→奇奇→奇偶" 依次为 1-2-3-4 类。
群延迟：$\alpha = \frac{N-1}{2}$，也是 $h(n)$ 的中心对称点。

由 $h(n)$ 求频率响应

求 $H(z)$、$H(e^{j\omega})$、幅度函数 $H(\omega)$ 的方法：
① 先提因子 $e^{-j\omega\frac{N-1}{2}}$
② 对称项两两合并，用欧拉公式消掉实部或虚部
③ 写成幅度函数 × 相位函数的形式

例题：第3类线性相位 FIR 求频响（点击展开）

已知：$h(n) = \{1, -2, 0, 2, -1\}$（$N=5$，奇数），求频率响应 $H(e^{j\omega})$、幅度函数 $H(\omega)$ 和相位函数 $\theta(\omega)$。 Step 1｜验证对称性 → 确定类型
中心 $\alpha = \frac{N-1}{2} = 2$，检查 $h(n) = -h(N-1-n)$： $$h(0)=1,\; h(4)=-1 \implies h(0)=-h(4)\;\checkmark$$ $$h(1)=-2,\; h(3)=2 \implies h(1)=-h(3)\;\checkmark$$ $$h(2)=0 \implies \text{中心为零}\;\checkmark$$ → 奇对称 + N 为奇数 → 第 3 类线性相位 Step 2｜写出 $H(z)$ $$H(z) = 1 - 2z^{-1} + 0\cdot z^{-2} + 2z^{-3} - z^{-4}$$ Step 3｜代入 $z=e^{j\omega}$，以中心 $\alpha=2$ 为基准重新分组 $$H(e^{j\omega}) = e^{-j2\omega}\Big[e^{j2\omega} - 2e^{j\omega} + 0 + 2e^{-j\omega} - e^{-j2\omega}\Big]$$ Step 4｜利用奇对称性 $h(\alpha+k) = -h(\alpha-k)$ 合并
将关于中心对称的项配对： $$\bigl[e^{j2\omega} - e^{-j2\omega}\bigr] + \bigl[-2e^{j\omega} + 2e^{-j\omega}\bigr]$$ 由欧拉公式 $e^{jx} - e^{-jx} = 2j\sin x$： $$= 2j\sin(2\omega) + 2 \cdot 2j\sin(\omega) = 2j\bigl[\sin(2\omega) + 2\sin(\omega)\bigr]$$ Step 5｜提取 $j = e^{j\pi/2}$，整理为幅-相形式 $$H(e^{j\omega}) = e^{-j2\omega} \cdot 2j\bigl[\sin(2\omega) + 2\sin(\omega)\bigr]$$ $$= \underbrace{2\bigl[\sin(2\omega) + 2\sin(\omega)\bigr]}_{H(\omega)} \cdot \underbrace{e^{-j(2\omega - \pi/2)}}_{\text{相位因子}}$$ 结论：
  • 幅度函数：$H(\omega) = 2\sin(2\omega) + 4\sin(\omega)$（纯实数，可正可负）
  • 相位函数：$\theta(\omega) = -\omega\dfrac{N-1}{2} - \dfrac{\pi}{2} = -2\omega - \dfrac{\pi}{2}$ ✓（第 3 类特征）
  • $\omega=0$（直流）时：$H(0)=0$ → 第 3 类在直流处必为零，不能做低通
  • $\omega=\pi$ 时：$H(\pi)=0$ → 高频处也为零
通用公式（第 3 类）：
$N$ 为奇数，$h(n) = -h(N-1-n)$，中心 $h(\alpha)=0$，则： $$H(\omega) = 2\sum_{k=1}^{(N-1)/2} h(\alpha + k)\sin(\omega k)$$

5.3 零点特性与滤波器类型判断

四零点组：如果 $z_i$ 是零点，则以下也是零点：
$z_i^*$（共轭）、$1/z_i$（倒数）、$1/z_i^*$（共轭倒数）
在实轴或单位圆上可能有重合。

由零点判断滤波器类型

零点位置含义滤波器类型
$z=1$（低频）有零点低频阻断 → 高频通过高通
$z=-1$（高频）有零点高频阻断 → 低频通过低通
两者都有零点高低频都阻断带通
两者都无零点高低频都通过带阻

第6章 数字信号处理系统的实现

IIR 滤波器的网络结构 必考

流图的三种基本运算单元：加法器、乘法器（标量）、延迟器（$z^{-1}$）。
画流图时，$z^{-1}$ 项的系数必须为 1。若差分方程中某项系数不为 1，需分子分母同除以该系数化为标准形式。
★ 直接 I 型结构
  • 上下各一条延迟链：上面对应分子（前馈），下面对应分母（反馈）
  • 延迟单元数 = 分子阶次 + 分母阶次
  • 箭头方向：全部向右
  • 分子系数直接对应；分母系数取相反数
★ 直接 II 型（正准型 / Canonic）结构 重点
  • 将直接 I 型的两条延迟链合并为中间一条
  • 延迟单元数 = max(分子阶次, 分母阶次)（最节省延迟单元）
  • 左半部分为分母（反馈），右半部分为分子（前馈）
  • 反馈方向向左，前馈方向向右
直接型的缺点：零极点控制难——每个系数的改变都会影响所有零极点分布。对系数量化敏感，阶次高时影响更大。
★ 级联型（串联型）结构
  • 将 $H(z)$ 分解为一阶/二阶因式相乘：$H(z) = \prod H_k(z)$
  • 每个基本节用正准型结构，串联连接
  • 优点：零点和极点可以单独控制调整
  • 注意：各基本节顺序可调换，结构不唯一
★ 并联型结构
  • 将 $H(z)$ 做部分分式展开：$H(z) = \sum H_k(z)$
  • 各基本节并列连接，输出相加
  • 优点：极点易控制；可并行运算，速度快
  • 缺点：二阶节的零点不是系统零点，零点不易控制
三种 IIR 结构对比：
结构零极点控制运算速度量化误差
直接型难（系数间耦合大）一般最大
级联型零极点均可单独控制一般居中
并联型极点可控，零点不可控最快（并行）最小

统一示例：$H(z) = \dfrac{1 + 2.5z^{-1} + z^{-2}}{1 - 0.75z^{-1} + 0.125z^{-2}}$

★ FIR 结构信号流图示例（点击展开）

统一示例：$h(n) = \{1, 2, 3, 2, 1\}$（偶对称，$N=5$，第1类线性相位）
$H(z) = 1 + 2z^{-1} + 3z^{-2} + 2z^{-3} + z^{-4}$

① 横截型（直接型）

5 个抽头、5 个乘法器、4 个延迟单元。结构简单直接。

② 线性相位型 重点

利用 $h(n)$ 偶对称性，$h(0)=h(4), h(1)=h(3)$，对称项先相加再乘系数 → 只需 3 个乘法器！

★ 横截型（直接型）结构
  • 上面一条延迟链，$N$ 长序列需 $N-1$ 个延迟单元
  • 每个抽头乘以对应系数 $h(0), h(1), \ldots, h(N-1)$
  • 所有乘法器输出相加得 $y(n)$
级联型结构

将 $H(z)$ 分解为二阶因式相乘，每个二阶节用直接型画出，串联连接。

★ 线性相位型结构 必考画图

核心优势：乘法器次数可以减少近一半！

利用 $h(n)$ 的对称性（偶对称/奇对称），将对称项合并：

  • 偶对称：$h(n) = h(N-1-n)$，对称项直接相加后乘一次系数
  • 奇对称：$h(n) = -h(N-1-n)$，对称项先取反再合并（引入乘 $-1$）
  • 中间项（若有）单独处理
线性相位 FIR 滤波器分类：
类型N对称性特征
第 I 类奇数偶对称最常用
第 II 类偶数偶对称$H(\pi)=0$，不能做高通
第 III 类奇数奇对称$h(\frac{N-1}{2})=0$
第 IV 类偶数奇对称$H(0)=0$
频率采样型结构
  • 由两部分组成：梳状滤波器 + 一阶谐振器并联
  • 梳状滤波器：$H_1(z) = 1 - z^{-N}$，差分方程 $y(n) = x(n) - x(n-N)$
  • 频响 $|H_1(e^{j\omega})| = 2|\sin\frac{N\omega}{2}|$（梳齿状）
  • 特别适用于窄带滤波器（如雷达信号处理），可节约运算量
  • 有反馈（谐振器部分是 IIR 一阶结构）
不需要画频率采样型流图，但要知道其组成和特点。

有限字长效应

三种量化误差来源
来源说明
A/D 转换采样后叠加量化噪声
系数量化滤波器系数用有限位表示，改变零极点位置
运算量化乘法运算引入舍入噪声（主要指乘法器）
A/D 转换量化效应
  • 可建模为：理想采样 + 量化噪声 $e(n)$
  • 舍入处理：均值 = 0，方差 $\sigma_e^2 = \frac{q^2}{12}$，其中 $q = 2^{-B}$
  • $B$ = 有效尾数位数（不含符号位）
★ 量化信噪比结论（必记）：有效位数每增加 1 位，量化信噪比约增加 6 dB。
运算量化误差（乘法舍入噪声）

输出噪声方差公式：

$$\sigma_{y}^2 = \sigma_e^2 \sum_{n=0}^{\infty} |h_i(n)|^2 = \frac{q^2}{12} \sum |h_i(n)|^2$$

其中 $h_i(n)$ 为第 $i$ 个噪声源到输出的后续子网络的单位脉冲响应。

三种 IIR 结构的输出噪声大小比较：
直接型（最大） > 级联型（居中） > 并联型（最小）
原因：并联型的每个噪声源直接到输出端，不经过其他子网络放大。
FIR 滤波器输出噪声（简单！）
$$\sigma_y^2 = M \cdot \frac{q^2}{12}$$

$M$ = 乘法器个数。因为 FIR 滤波器的所有量化噪声直接加到输出端（非递归），只需数乘法器个数即可。

系数量化与极点灵敏度
  • 系数量化对滤波器的影响与字长和结构有关
  • 极点位置灵敏度：各极点位置对系数偏差的敏感程度
  • 极点位置灵敏度与极点间距离成反比
  • 并联型/级联型灵敏度小，直接型灵敏度大
结论：高阶滤波器应避免使用直接型，应分解为低阶结构的并联或级联形式。
极限环振荡
  • 零输入极限环振荡：信号归零后，由于舍入引起的非线性，输出不趋于 0，而停留在某个值或在固定值间振荡
  • 大信号极限环振荡（溢出振荡）：定点补码加法运算中产生溢出，大信号时引起系统振荡
